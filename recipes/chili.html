Description

Chili

Ingredients

  • ground beef
  • tomato sauce
  • beans
  • onion
  • garlic
  • chili powder
  • oregano
  • cumin
  • chili pepper
  • celery
  • tomato paste
  • chili pepper

Instructions

  1. brown ground beef
  2. add onion and garlic
  3. add tomato sauce
  4. add beans
  5. add chili powder
  6. add oregano
  7. add cumin
  8. add chili pepper
  9. add celery
  10. add tomato paste
  11. add chili pepper
  12. cook for 1 hour
  13. serve

Notes

Make sure to add the chili pepper before the tomato paste so it doesn't get soggy.
